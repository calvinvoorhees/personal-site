// 01. Applications

Color Conform Studio

Automated node-based color management + conform tool designed with colorists in mind. Analyzes camera metadata, automatically groups footage by camera type, and applies the appropriate IDT via a saved .drx grade. Fully customizable node tree application. Designed for DaVinci Resolve Studio.
